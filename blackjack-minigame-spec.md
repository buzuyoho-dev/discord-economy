# 블랙잭 미니게임 구현 스펙

## 1. 개요
- 명령어: `/블랙잭 [베팅금액]`
- 플레이어 1명 vs 하우스(딜러) 1대1 대결
- '미니게임' 카테고리의 하위 종류로 설계 (추후 다른 미니게임 추가 대비)
- 미니게임은 수수료(정산 시스템의 5% 세금) 없음. 패배 시 베팅금은 하우스로 완전 흡수.

## 2. 확정된 게임 룰
| 항목 | 내용 |
|---|---|
| 자연승(카드 2장으로 21) 배당 | 1.5배 (예: 1000 베팅 → 승리 시 +1500) |
| 더블다운 (double down, 카드 2장 받은 후 베팅금 2배로 올리고 카드 1장만 더 받는 선택지) | ❌ 없음 |
| 스플릿 (split, 같은 숫자 카드 2장을 2개 손으로 나눠 진행) | ❌ 없음 |
| 플레이어 액션 | 히트(hit, 카드 1장 더 받기) / 스탠드(stand, 그만 받고 승부) 만 존재 |
| 딜러 AI | 자기 패 합 17 이상이 될 때까지 무조건 히트, 17 이상이면 무조건 스탠드 (소프트 17 예외 규칙 없음) |
| 무승부(push, 점수가 같음) | 베팅금 그대로 환급, 승패 카운트에 영향 없음 |
| 최소 베팅금 | 100,000포인트 이상 |
| 최대 베팅금 | 보유 포인트의 25% (`Math.floor(보유포인트 * 0.25)`로 내림 처리) |
| 일일 플레이 제한 | **블랙잭 게임에 한해** 하루 5회 (KST 자정 초기화, 기존 `kstMidnightUtc` 헬퍼 재사용) |
| 액션 타임아웃 | 히트/스탠드 버튼 60초 미응답 시 자동 스탠드 처리. 이 경우도 플레이 횟수는 정상 차감 (어뷰징 방지) |
| 서버 재시작 시 진행 중이던 게임 | 리스크는 하우스가 부담 (MVP 단계에서는 별도 복구 로직 없음) |
| 남은 플레이 횟수 표시 | 게임 결과 임베드(embed, 디스코드에서 카드 형태로 꾸며서 보여주는 메시지 박스) 하단에 "오늘 남은 횟수: N/5" 형태로 표시 |

## 3. DB 스키마 (Prisma)
```prisma
model MinigamePlayLog {
  id        String   @id @default(cuid())
  userId    String
  gameType  String   // "BLACKJACK", 추후 "RPS" 등 추가 예정
  count     Int      @default(0)
  playDate  DateTime // KST 자정 기준 날짜값

  @@unique([userId, gameType, playDate])
}
```
> 💡 유니크 제약(unique constraint, 이 조합의 값은 DB에 하나만 존재하도록 강제하는 규칙)이 `[userId, gameType, playDate]`로 걸려 있어서, 게임 종류(gameType)별로 카운트가 완전히 분리됨. 블랙잭 카운트와 향후 추가될 다른 미니게임 카운트는 서로 합산되지 않음.

## 4. 진행 중인 게임 상태 관리
- Railway 단일 인스턴스 배포 환경이므로 **인메모리 Map**으로 관리 (DB에 저장하지 않는 임시 저장소)
```js
// 💡 게임 진행 중인 유저의 카드 상태를 잠깐 기억해두는 저장소
const activeBlackjackGames = new Map(); // key: userId, value: { deck, playerHand, dealerHand, bet, message, timeout }
```
- 이미 진행 중인 게임이 있는 유저가 `/블랙잭` 재입력 시 → 거부 (동시 2판 방지)

## 5. 명령어 흐름
```
/블랙잭 [베팅금액]
  ↓ 검증: 오늘 블랙잭 5회 이내? / 베팅금 ≤ 보유포인트*25%? / 진행 중인 게임 없음?
  ↓ 베팅금 즉시 차감
  ↓ 카드 2장씩 배분 (딜러 1장은 비공개)
  ↓ 자연승(21) 체크 → 있으면 즉시 정산 후 종료
  ↓ 없으면 [히트] [스탠드] 버튼 표시 (60초 타임아웃 → 자동 스탠드)
  ↓ 히트 반복 → 21 초과(버스트) 시 즉시 패배 처리
  ↓ 스탠드 선택 → 딜러 자동 진행(17까지 히트) → 승패 비교 → 정산
  ↓ MinigamePlayLog(gameType: "BLACKJACK") count +1
  ↓ 결과 임베드에 "오늘 남은 횟수: N/5" 표시
```
